# 《赤穹协议》(Crimson Vault Protocol)

一款基于 Web 的多支线、多结局地缘政治文字冒险游戏。**v3.0** 加入动态剧情、供应链/军备/间谍模拟与 Roguelike 元素。

## 🌍 背景设定

公元 2049 年，五大拥核势力在毁灭边缘签署《赤穹协议》，禁止直接战争但允许代理人冲突。你扮演南太平洋联邦的国家安全顾问，在核威慑阴影下周旋于大国之间，决定国家命运。

## 🎮 核心玩法

- **动态核平衡系统**：每个决策影响全球核紧张指数，紧张度达 100 即核战结局
- **五大国关系矩阵**：美、中、俄、欧、印的好感、信任、威胁、贸易实时变化
- **多维行动**：外交谈判、代理人战争、科技研发、经济制裁、**派遣间谍**
- **多章节剧情**：南海危机 → 资源发现 → 非洲代理人 → 科技竞赛 → 间谍网络 → 危机升级 → 最终抉择
- **10+ 结局**：从第六极崛起到核冬天、AI 统治、影子统治者等，你的选择塑造世界

### v3.0 新增

| 模块 | 说明 |
|------|------|
| **🤖 动态剧情** | 离线模板 + 种子随机生成剧情开场白，同一场景多版本呈现 |
| **📦 资源供应链** | 石油、稀土、钴、粮食、芯片依赖大国供应，关系恶化会断供并造成预算惩罚 |
| **🔭 军备竞赛** | 五大国军备等级每回合上升，部分天赋可查看各国军备 |
| **🕵️ 间谍博弈** | 派遣间谍渗透目标国，数回合后结算成功/失败，影响情报与关系 |
| **🗺️ 随机地图** | 每局随机 6 个区域（危机热点、资源富集、科技枢纽等），影响中期场景类型 |
| **⚡ 事件链** | 部分随机事件带后续节点（如石油危机 → 后续），形成连续剧情 |
| **⭐ Perk 系统** | 开局从 6 个随机天赋中选 2 个（预算大师、外交家、间谍加成、供应链专家等），影响整局 |

## ▶️ 快速开始

### 网页版（推荐）

```bash
cd chiqiong-protocol
python3 -m http.server 8081
# 浏览器访问 http://localhost:8081
```

### Python 命令行版

```bash
cd chiqiong-protocol
python3 main.py
```

> 注：Python 版为简化流程，完整玩法请使用网页版。

## 📁 项目结构

```
chiqiong-protocol/
├── index.html          # 网页版入口
├── main.py             # Python 命令行版
├── css/
│   ├── style.css       # 全局样式
│   └── game.css        # 游戏界面与 Roguelike 面板样式
├── js/
│   ├── roguelike-data.js  # 供应链/军备/Perk/剧情模板/随机地图数据
│   └── game.js         # 游戏引擎（状态、场景、事件、回合逻辑）
└── images/
    ├── flags/          # 国旗图标
    └── scenes/         # 场景插图
```

## 🖼️ 视觉与交互

- 响应式设计，支持桌面与移动设备
- SVG 矢量图形
- 状态栏展示：核紧张度、国家资源、五大国关系、秘密资产、**供应链状态**、**本局天赋**（及可选军备等级）
- 开局 Perk 选择界面；游戏中可点击「派遣间谍」部署任务

## 💾 游戏保存

- **网页版**：保存到浏览器 localStorage（键名 `chiqiongProtocolV3Save`，兼容旧版 `chiqiongProtocolV2Save`）
- **Python 版**：保存到 `savegame.json`
